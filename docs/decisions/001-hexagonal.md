# ADR-001: Hexagonal Architecture

## Estado
Aceptado

## Contexto
Se requiere una arquitectura mantenible y desacoplada.

## Decisión
Se adopta arquitectura hexagonal.

## Alternativas
- arquitectura en capas
- microservicios

## Consecuencias
- mayor complejidad inicial
- mejor escalabilidad

## Justificación
Permite desacoplar dominio de infraestructura.
