# Domain Overview

El dominio representa el núcleo del sistema y está completamente desacoplado de frameworks.

## Entidades principales

- User
- Student
- Teacher
- Course
- Group
- Schedule
- Enrollment
- AttendanceSession
- AttendanceRecord
- ProgressRecord

## Principio clave

Todo elemento del dominio pertenece a una institución.

## Objetivo

Representar las reglas reales del negocio, no la estructura de la base de datos.
